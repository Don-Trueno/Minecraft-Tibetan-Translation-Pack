# Minecraft 藏语语言包

[English](/readme.md) | 简体中文

这一语言包为Minecraft加入了新的语言——[藏语](https://zh.wikipedia.org/wiki/%E8%97%8F%E6%96%87)（`bo_bt`），供爱好者研究学习。


## 支持版本

**👍 推荐：**

- Minecraft 1.19.3+

1.13~1.19.2间也能使用本语言包，但特定游戏内容的翻译**可能会出错**，相关问题暂不考虑修复。

**❌️ 不支持**

- Minecraft 1.13-

这些版本并不使用JSON格式的语言文件。


## 如何使用

1. 前往[最新 Releases](https://github.com/Don-Trueno/Minecraft-Bod-yig-Language-Pack/releases/latest)页面。
2. 下载资源包。
3. 安装到你的游戏中。

不会安装？请参阅[wiki 教程](https://minecraft.wiki/w/Tutorial:Loading_a_resource_pack)。
